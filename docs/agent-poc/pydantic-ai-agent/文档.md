# Pydantic AI Agent

> 目标：生产级实现，AgentService + tools + structured output + FastAPI route

---

## 职责范围

```
POST /api/analyze?pipeline=agent
  → AgentService.run()
    → step1: 解析展平 (Python tool)
    → step2: DuckDB 入库
    → step3: 字段映射 (读上下文)
    → step4: SQL 指标计算
  → [确定性层: Threshold → Evidence → Report]
```

## Pydantic AI Agent Tool Use

| 名字                       | 类型     | 说明                                                                                        | 共用  |
| -------------------------- | -------- | ------------------------------------------------------------------------------------------- | ----- |
| AgentService               | 编排器   | 控制流程、调用 Agent、串联工具                                                              | False |
| Agent.md                   | 流程指引 | 写任务边界、允许工具、输出格式、失败策略                                                    | False |
| `read_workspace_file`      | tool     | 读本次上传文件                                                                              | True  |
| `write_workspace_file`     | tool     | 写脚本 / manifest / 中间结果                                                                | True  |
| `run_python_script`        | tool     | 在 workspace 沙箱内执行 Python 脚本，限制路径 / 超时 / 内存 / 网络                          | True  |
| `duckdb_tool`              | tool     | 建临时库、注册 parquet、执行 SQL                                                            | True  |
| `read_context_tool`        | tool     | 读指标文档、字段定义、行业规则                                                              | True  |
| `validate_result_tool`     | tool     | 校验 manifest / mapping / metrics                                                           | True  |
| `profile_table_tool`       | tool     | 查看表结构、列名、样本、空值率                                                              | True  |
| `list_workspace_files`     | tool     | 查看当前目录文件                                                                            | True  |
| `submit_final_result_tool` | tool     | 强制提交标准 AgentResult :{"tables_manifest":"...","mapping":[],"metrics":[],"warnings":[]} | False |

## 工具定义

```python
# flatten_tool: 输入原始数据 → 输出 DataFrame JSON
# db_tool: 输入 SQL + 表名 → 输出查询结果 JSON
# mapping_tool: 输入 profiles + 上下文文档 → 输出 SemanticMapping[]
```

## FastAPI 集成

- 路由：`POST /api/analyze?pipeline=agent`
- SSE 日志复用现有 `GET /api/stream/{report_id}`
- 输出格式兼容现有 `AnalyzeResponse`

## 评判标准

| 维度           | 期望                                                                               |
| -------------- | ---------------------------------------------------------------------------------- |
| 嵌套 JSON 展平 | 5 层以上深度也能稳定展平                                                           |
| 生成 SQL       | 覆盖 ratio / period_change / share_by_dimension / concentration / top_contribution |
| DeepSeek 兼容  | function calling 正常，tool result 回传不出错                                      |
| FastAPI 接入   | routes / SSE / 异常处理能复用现有代码                                              |
| 字段映射       | 读指标计算文档.md 后匹配准确率 > 90%                                               |
